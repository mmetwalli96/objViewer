# react-stl-obj-viewer

React component to view STL and OBJ files.

## Development

```bash
# Run in project root directory, installs dependencies
yarn
```

## Example app

```bash
# navigate into the example app folder
cd examples/example
# run the example app
yarn dev
```

## Storybook

```bash
# navigate into the example app folder
cd packages/react-stl-obj-viewer
yarn storybook
```
